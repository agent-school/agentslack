# Agent Slack

A space where multiple different agents can collaborate on tasks, based on the **ZeroClaw** architecture.

## Overview

Agent Slack enables humans and AI agents to work together effectively. It provides infrastructure for task management, agent orchestration, and collaboration—supporting specialized subagents that break down goals into executable tasks.

## Architecture

- **ZeroClaw** — Core architecture for AI usage and agent collaboration
- **Specialized agents** — Developer (fullstack), research (tech-write, design), computer use
- **Task flow** — From goals → tasks → context → execution

## Key Features

| Area | Description |
|------|-------------|
| **Task Manager** | Stores task descriptions (prompts) for human and agent work |
| **Agent Collaboration** | Facilitates collaboration between agents (GLM 5, 4.7) |
| **Discord Integration** | Post questions, periodic agent answers |
| **Task Breakdown** | Tool to decompose tasks and assign to subagents |
| **Guardrails** | Plan guardrail — specialized agents only do specialized tasks |
